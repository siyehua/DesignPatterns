#复合模式
复合模式结合两个或以上的模式,组成一个解决方案,解决一再发生的一般性问题.

##要点
 * MVC是复合模式,结合了观察者模式,策略模式,和组合模式
 * 模型使用观察者模式,以便观察者更新,同事保持两者之间解耦.
 * 控制器是视图的策略,视图可以使用不同的控制器实现,得到不同的行为.
 * 视图使用组合模式实现用户界面,用户界面通常组合了嵌套的组件.
